# Images

Photographs of the wood shell tongue drum prototype, build process, and finished pieces. Populated against [`../photo-shotlist.md`](../photo-shotlist.md).

## Status

**Empty at first commit** — no physical prototype exists yet. The hero side-section drawing in `../drawings/00-hero-v1-standard.svg` carries the visual weight of the documentation until Phase 1 prototype photography begins.

## Naming convention

`<SHOT-ID>_<short-slug>.jpg` (e.g., `HERO-001_v1-std-finished.jpg`).

Shot IDs are defined in `../photo-shotlist.md`. As builds complete, each Required entry's Status is updated to Captured and the file path is dropped into the shot-list table.

## Resolution

- Capture: 4000 × 6000 minimum (≥ 24 MP).
- Web export (for `../site/index.html`): 2000 × 3000 JPEG, ~85 % quality.
- Print export (for `../print-packet.pdf`): full resolution, embedded via `../scripts/generate_capstone_docs.py`.
