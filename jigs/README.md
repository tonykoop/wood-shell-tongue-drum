# Jigs

This folder tracks fixture intent without pretending the physical jigs are already built.

## Required Jigs

1. Stave bevel reference jig for 11.25 degree cylinder staves.
2. Shell glue-up support or caul that preserves roundness through cure.
3. Soundboard layout overlay and hold-down clearance template.
4. Temporary tuning / measurement support that does not damp the tongue field.

## Future Jigs

- Dome flip jig for V2 and V4.
- Hemisphere ring-stack centering fixture for V3 and V4.
- Gu-port drilling locator if repeated family builds justify it.
