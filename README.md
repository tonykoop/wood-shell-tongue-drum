# Wood Shell Tongue Drum

Engineering documentation and parametric design table for the wood-shell tongue drum — slit-tongue idiophone with a wooden resonant body.

Part of the [tonykoop/instrument-maker](https://github.com/tonykoop/instrument-maker) catalogue.
