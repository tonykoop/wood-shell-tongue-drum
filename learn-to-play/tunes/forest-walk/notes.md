# Notes: Forest Walk

**Composition:** Original tune written for the wood-shell tongue drum songbook.
MIT licensed — part of the Heifer Zephyr collection.
Designed for A3 ding (Standard 16" model); adapt if using Travel or Floor Pouf sizes.

**Range check:** A3 (ding)–E5 (high field). Uses the full range of the Standard model.

**Practice tips:**
1. Begin alone with the ding pickup — establish the tonal center before the melody starts.
2. Each 4-bar phrase is self-contained: learn them individually before connecting.
3. The D minor arpeggio in phrase 4 (A D F A) can be played as block chords for a final flourish.
