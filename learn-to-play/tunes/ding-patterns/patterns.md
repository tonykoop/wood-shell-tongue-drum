# Ding Drone Patterns ★☆☆

Use the ding as a sustained anchor point. Strike it, then add tongue-field notes above it.

## Pattern 1: Ding + fifth (for A3 ding, the fifth is E4)
```
Strike: A (ding)
Wait 2 beats
Strike: E
Wait 2 beats
Repeat
```

## Pattern 2: Ding + scale walk
```
Strike: A (ding)
Then slowly walk up: B C D E F G A
Pause at each note to hear the ding's resonance blend with it
Walk back down
```

## Pattern 3: Call and response
Strike the ding (A3). Listen. Then answer with a single higher note.
Let the ding fade before answering. This is meditation practice, not rhythm.

**Tip:** The ding is not just the lowest note — it is the spiritual anchor of the drum.
In handpan tradition, you return to the ding after each phrase. Use the same concept here.
